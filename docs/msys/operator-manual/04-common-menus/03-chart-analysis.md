# 차트 분석

## 1. 메뉴 접속 방법

- **경로**: 상단 메뉴 → 차트 분석
- **URL**: `/chart_analysis`
- **필요 권한**: `analysis`

## 2. 화면 구성

| 번호 | 영역 | 설명 |
|------|------|------|
| ① | 차트 영역 | 수집 데이터 시각화 |
| ② | 필터 패널 | 기간, Job ID 선택 |

## 3. 주요 기능

### 3.1 차트 조회

**조작 절차:**
1. 필터에서 기간/Job ID 선택
2. 조회 버튼 클릭
3. 차트 확인
