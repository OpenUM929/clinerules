# 원본 데이터

## 1. 메뉴 접속 방법

- **경로**: 상단 메뉴 → 원본 데이터
- **URL**: `/raw_data`
- **필요 권한**: `raw_data`

## 2. 화면 구성

| 번호 | 영역 | 설명 |
|------|------|------|
| ① | 데이터 테이블 | 원본 수집 데이터 |
| ② | 필터 | 날짜/Job ID 필터 |

## 3. 주요 기능

### 3.1 원본 데이터 조회

**조작 절차:**
1. 필터 조건 입력
2. 조회 버튼 클릭
