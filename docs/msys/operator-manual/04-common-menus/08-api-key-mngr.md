# API 키 관리

## 1. 메뉴 접속 방법

- **경로**: 상단 메뉴 → API 키 관리
- **URL**: `/api_key_mngr`
- **필요 권한**: `api_key_mngr`

## 2. 화면 구성

| 번호 | 영역 | 설명 |
|------|------|------|
| ① | 목록 테이블 | API 키 목록 |
| ② | 검색/필터 | 코드/담당자 검색 |
| ③ | 등록/수정 버튼 | API 키 등록/수정 |

## 3. 주요 기능

### 3.1 API 키 조회

**조작 절차:**
1. 검색어 입력 또는 필터 선택
2. 조회 버튼 클릭

### 3.2 API 키 등록/수정

**조작 절차:**
1. 등록 버튼 클릭
2. 코드, 만료일, 담당자 이메일 입력
3. 저장 버튼 클릭
