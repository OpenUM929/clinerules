# 대시보드

## 1. 메뉴 접속 방법

- **경로**: 상단 메뉴 → 대시보드
- **URL**: `/`
- **필요 권한**: `dashboard`

## 2. 화면 구성

<img src="images/dashboard-status-panel.png" width="800">

| 번호 | 영역 | 설명 |
|------|------|------|
| ① | 상태 요약 패널 | 전체 수집 작업 상태 요약 |
| ② | 알림 영역 | 만료 임박 API 키, 이상 작업 알림 |
| ③ | 차트 영역 | 수집 현황 차트 |

## 3. 주요 기능

### 3.1 상태 요약 확인

**목적:** 전체 수집 작업의 성공/실패/진행 중 상태를 한눈에 확인합니다.

**확인 방법:**
- 상태별 카운트 확인
- 클릭 시 해당 상태의 작업 목록으로 이동

## 4. 모니터링 체크리스트

- [ ] 성공률이 95% 이상인지 확인
- [ ] 실패 작업이 30분 이상 지속되지 않는지 확인
