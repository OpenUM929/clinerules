# 수집 스케줄

## 1. 메뉴 접속 방법

- **경로**: 상단 메뉴 → 수집 스케줄
- **URL**: `/collection_schedule`
- **필요 권한**: `collection_schedule`

## 2. 화면 구성

<img src="images/collection-schedule-filter.png" width="800">

| 번호 | 영역 | 설명 |
|------|------|------|
| ① | 필터 패널 | 날짜, Job ID, 상태 필터 |
| ② | 상태 뱃지 | 작업 상태 시각적 표시 |
| ③ | 상세 보기 버튼 | 작업 상세 정보 팝업 |

## 3. 주요 기능

### 3.1 작업 상태 필터링

**조작 절차:**
1. 필터 패널에서 조건 선택
2. 조회 버튼 클릭

### 3.2 작업 상세 확인

**조작 절차:**
1. 대상 행의 상세 보기 버튼 클릭
2. 팝업에서 수집 이력, 로그 확인
