# 관리자 (통계/템플릿)

## 1. 메뉴 접속 방법

- **경로**: 상단 메뉴 → 관리자
- **URL**: `/admin`
- **필요 권한**: `admin`

## 2. 화면 구성

| 번호 | 영역 | 설명 |
|------|------|------|
| ① | 통계 탭 | 메뉴 접근 통계 |
| ② | 템플릿 탭 | 엑셀 템플릿 관리 |

## 3. 주요 기능

### 3.1 통계 조회

**조작 절차:**
1. 기간/메뉴 선택
2. 조회 버튼 클릭
3. 차트/테이블 확인

### 3.2 엑셀 템플릿 관리

**조작 절차:**
1. 템플릿 탭 선택
2. 업로드/다운로드/삭제 버튼 사용
