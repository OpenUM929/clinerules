# 데이터 분석

## 1. 메뉴 접속 방법

- **경로**: 상단 메뉴 → 데이터 분석
- **URL**: `/data_analysis`
- **필요 권한**: `analysis`

## 2. 화면 구성

| 번호 | 영역 | 설명 |
|------|------|------|
| ① | 검색 패널 | 조건 기반 데이터 검색 |
| ② | 결과 테이블 | 검색 결과 표시 |

## 3. 주요 기능

### 3.1 데이터 검색

**조작 절차:**
1. 검색 조건 입력
2. 검색 버튼 클릭
3. 결과 확인
