# 외부 연동

## 1. 메뉴 접속 방법

- **경로**: 상단 메뉴 → Airflow / Kafka UI
- **URL**: 외부 URL (새 창)
- **필요 권한**: 해당 메뉴 권한

## 2. 외부 시스템

| 메뉴 | 설명 | URL |
|------|------|-----|
| Airflow | 워크플로우 스케줄러 | 외부 링크 |
| Kafka UI | 메시지 브로커 UI | 외부 링크 |

## 3. 주의사항

- 외부 시스템은 별도 로그인 필요할 수 있음
- 네트워크 접근 권한 확인 필요
