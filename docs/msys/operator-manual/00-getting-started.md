# 빠른 시작 가이드

## 1. 시스템 접속

1. 브라우저에서 `http://[서버주소]:18080` 접속
2. 로그인 화면에서 ID/PW 입력
3. 관리자 계정으로 접속 시 전체 메뉴 확인 가능

## 2. 필수 확인 사항

| 항목 | 확인 방법 |
|------|----------|
| 서비스 기동 상태 | `ps -ef \| grep msys` |
| DB 연결 상태 | 대시보드 데이터 로드 여부 |
| 로그 확인 | `/data/external_data_monitoring/log/` |

## 3. 기본 조작 순서

### 일일 업무
1. 대시보드 접속 → 수집 현황 확인
2. 수집 스케줄 → 이상 작업 확인
3. API 키 관리 → 만료 임박 키 확인

### 주간 업무
1. 관리자 설정 → 사용자 승인 대기 확인
2. 통계 → 메뉴 접근 현황 확인
