### 변경 요약 (1줄, 50자 이내)
> 배치 CSV 스트리밍·Staging DB·배치 명칭 기능 추가

### 변경 배경 / 이유
> 1.9만 명 규모 배치 처리 시 메모리 초과 방지를 위해 Staging DB(SQLite) 기반 스트리밍 처리 도입. 동시에 배치 식별을 위한 사용자 지정 명칭(display_name) 기능 추가.

### 변경 전 → 변경 후
- Before: 배치 처리 시 전체 CSV를 메모리에 로드, 배치 구분은 batch_id(날짜형식)만 가능
- After : Staging DB(batch_dir/staging.db)로 스트리밍, 배치 생성·그룹분석에서 사용자 명칭 지정/수정 가능

### 영향 범위
> 배치 처리 전 흐름(업로드→처리→완료), 그룹분석 배치 이력 패널, 메타데이터 배치 목록
