### 변경 사항 요약
- 일괄 수정 팝업 사용자 경험 개선 및 버그 수정

### 상세 변경 사항

**1. 일괄 수정 팝업 기본값 설정**
- 등록일: 오늘 날짜 자동 입력
- 기간: 1년 기본값 설정

**2. 이메일 입력 UI 개선**
- 선택된 데이터의 이메일 중 선택 또는 새로 입력 가능
- select + input 조합 UI로 변경

**3. 팝업 너비 고정**
- 750px로 고정하여 체크박스 토글 시 크기 변경 방지

**4. Shift+클릭 범위 선택 수정**
- 체크박스 셀 클릭 시에도 범위 선택 정상 작동

**5. 팝업 라벨 클릭 토글 기능**
- 팝업 내 라벨(기간, 등록일, 이메일) 클릭 시 체크박스 토글
