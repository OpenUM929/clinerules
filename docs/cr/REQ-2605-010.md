### 변경 전 → 변경 후
- Before: 테이블 헤더 "26.05" 아래에 25.12 데이터 표시, 주간 차트 X축 미표시
- After : 모든 화면 과거→최신(25.12→26.05) 일관 정렬, YY.MM.주차 레이블 표시
