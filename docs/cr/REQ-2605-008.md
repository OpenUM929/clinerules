### 변경 배경 / 이유
- 세부 그룹 pill의 `title` 속성이 18자로 자른 `subDisplayName`을 그대로 사용
- 화면 표시는 18자 제한 유지가 맞지만, 툴팁은 원본 전체 이름을 보여주어야 사용자가 정보를 정확히 파악 가능

### 변경 전 → 변경 후
- Before: `subGroupPill.title = subDisplayName` (18자로 잘린 이름)
- After : `subGroupPill.title = subTooltipName` (원본 전체 이름)

### 영향 범위
- `static/js/pages/collection_schedule.js`: 세부 그룹 툴팁만 영향, 다른 로직 영향 없음
