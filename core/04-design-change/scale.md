# 변경 규모 판단

"전체 디자인 변경", "XX 디자인으로 바꿔줘", "wordcloud 디자인 적용" 등의 요청이 들어오면 **반드시** 디자인 변경 규칙을 최우선으로 적용한다.

| 규모 | 조건 | 적용 절차 |
|------|------|----------|
| **경량** | 단일 요소 수정 (버튼 색상, 텍스트, 아이콘 1개 등) | [경량 절차](light.md) |
| **표준** | 복수 요소 수정, 레이아웃 변경, 페이지 단위 변경 | [표준 절차](standard.md) |
| **전체** | "전체 디자인 변경", "XX 디자인으로 통일" 등 | 표준 절차 + 영향 페이지 전수 목록화 |
