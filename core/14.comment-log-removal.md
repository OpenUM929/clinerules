# 14. 주석 및 로그 제거 작업 규칙

## 작업 유형
- 소스 코드 주석(`#`, `//`, `<!-- -->`, `--`, `/* */`) 일괄 제거
- 디버그 로그(`console.log`, `print`, `logger.debug` 등) 제거
- **Python docstring(`"""..."""`)은 보존**

## 핵심 원칙 (필수)
1. **제외 디렉토리 절대 준수**
   - `static/vendor/` - 외부 라이브러리 (minified 파일 손상 방지)
   - `msys_venv/` - 가상환경 (패키지 손상 방지)
   - `node_modules/`, `__pycache__/`, `.git/`, `log/`, `document/`

2. **파일 패턴 제외**
   - `.min.js`, `.min.css`, `.umd.js` (압축 파일 보호)

3. **언어별 안전한 처리 방식**
   - **Python**: `tokenize` 모듈 사용 (docstring 보존, 문자열 보호)
   - **JS/HTML/SQL/CSS**: 정규식 기반 제거 (문자열 보호 로직 포함)

## 작업 절차 (체크리스트)
- [ ] 1. 백업 생성: `backup_YYYYMMDD_HHMMSS/` 디렉토리
- [ ] 2. 스크립트 작성: `remove_comments_safe.py` (하단 템플릿 참조)
- [ ] 3. Dry-run 테스트: 단일 파일 대상 검증
- [ ] 4. 전체 실행 (타임아웃 충분히 설정: 300초 이상)
- [ ] 5. 검증:
      - Python: `python -m py_compile <file>`
      - App import: `from msys_app import create_app`
      - JS: 브라우저 콘솔 에러 확인
- [ ] 6. Git diff 검토: `git diff --stat`, `git diff static/vendor/`

## 안전한 스크립트 템플릿 (remove_comments_safe.py)

```python
#!/usr/bin/env python3
"""
주석 제거 스크립트 - 안전 버전
- static/vendor/, msys_venv/ 절대 제외
- .min.js, .min.css 제외
- Python: tokenize 모듈 사용 (docstring 보존)
"""

import os
import re
import io
import tokenize
import shutil
from pathlib import Path
from datetime import datetime

# 제외 디렉토리 (부분 일치 포함)
EXCLUDE_PATTERNS = [
    'node_modules', 'msys_venv', '__pycache__', '.git',
    'static/vendor', 'log', 'document', 'backup_', '.opencode'
]

# 제외 파일 패턴
EXCLUDE_FILE_PATTERNS = ['.min.js', '.min.css', '.umd.js']

def should_exclude(path):
    path_norm = str(path).replace('\\', '/')
    return any(p in path_norm for p in EXCLUDE_PATTERNS)

def should_exclude_file(filename):
    return any(filename.endswith(p) for p in EXCLUDE_FILE_PATTERNS)

def read_file_safe(filepath):
    for enc in ['utf-8', 'cp949', 'euc-kr']:
        try:
            with open(filepath, 'r', encoding=enc) as f:
                return f.read(), enc
        except:
            continue
    return None, None

def write_file_safe(filepath, content, encoding='utf-8'):
    try:
        with open(filepath, 'w', encoding=encoding) as f:
            f.write(content)
        return True
    except:
        return False

def remove_python_comments(content):
    """Python: tokenize 모듈로 주석만 정확히 제거"""
    try:
        tokens = list(tokenize.generate_tokens(io.StringIO(content).readline))
        new_tokens = [t for t in tokens if t.type != tokenize.COMMENT]
        return tokenize.untokenize(new_tokens)
    except:
        return content

def remove_js_comments_simple(content):
    """JS: 단순 주석 제거"""
    content = re.sub(r'/\*.*?\*/', '', content, flags=re.DOTALL)
    lines = content.split('\n')
    new_lines = [line[:line.find('//')].rstrip() if '//' in line else line for line in lines]
    return '\n'.join(new_lines)

def remove_html_comments(content):
    return re.sub(r'<!--.*?-->', '', content, flags=re.DOTALL)

def remove_sql_comments_simple(content):
    content = re.sub(r'/\*.*?\*/', '', content, flags=re.DOTALL)
    lines = content.split('\n')
    new_lines = [line[:line.find('--')].rstrip() if '--' in line else line for line in lines]
    return '\n'.join(new_lines)

def remove_css_comments_simple(content):
    return re.sub(r'/\*.*?\*/', '', content, flags=re.DOTALL)

def process_file(filepath):
    if should_exclude_file(Path(filepath).name):
        return False
    
    content, encoding = read_file_safe(filepath)
    if content is None:
        return False
    
    original = content
    ext = Path(filepath).suffix.lower()
    
    if ext == '.py':
        content = remove_python_comments(content)
        encoding = 'utf-8'
    elif ext == '.js':
        content = remove_js_comments_simple(content)
    elif ext == '.html':
        content = remove_html_comments(content)
    elif ext == '.sql':
        content = remove_sql_comments_simple(content)
    elif ext == '.css':
        content = remove_css_comments_simple(content)
    else:
        return False
    
    if content != original and write_file_safe(filepath, content, encoding):
        return True
    return False

def create_backup():
    timestamp = datetime.now().strftime('%Y%m%d_%H%M%S')
    backup_dir = Path(f'backup_{timestamp}')
    backup_dir.mkdir(exist_ok=True)
    
    dirs_to_backup = ['dao', 'service', 'routes', 'mapper', 'models', 'msys', 'utils', 'sql', 'DDL', 'templates', 'static/js', 'static/css']
    for d in dirs_to_backup:
        src = Path(d)
        if src.exists():
            shutil.copytree(src, backup_dir / d, ignore=shutil.ignore_patterns('*.min.js', '*.min.css', 'vendor'))
    
    print(f"백업 위치: {backup_dir}")
    return backup_dir

def main():
    print("주석 제거 스크립트 - 안전 버전")
    print("=" * 60)
    
    # 백업
    print("\n1. 백업 생성 중...")
    backup_dir = create_backup()
    
    # 처리
    print("\n2. 주석 제거 처리 중...")
    processed = modified = 0
    for root, dirs, files in os.walk('.'):
        dirs[:] = [d for d in dirs if not should_exclude(os.path.join(root, d))]
        for file in files:
            filepath = os.path.join(root, file)
            if should_exclude(filepath):
                continue
            ext = Path(filepath).suffix.lower()
            if ext in ('.py', '.js', '.html', '.sql', '.css'):
                processed += 1
                try:
                    if process_file(filepath):
                        modified += 1
                        print(f"  처리됨: {filepath}")
                except Exception as e:
                    print(f"  오류 {filepath}: {e}")
    
    print(f"\n완료: {processed}개 검사, {modified}개 수정")
    print(f"백업 위치: {backup_dir}")

if __name__ == '__main__':
    main()
```

## 장애 사례 및 복구

### 사례 1: jinja2 패키지 손상
- **증상**: `re.error: bad character range` (e.txt)
- **원인**: `msys_venv/` 내부 파일 수정
- **해결**: 가상환경 재생성 + 의존성 재설치
  ```bash
  # Windows PowerShell
  cd D:\dev\msys
  Remove-Item -Recurse -Force msys_venv
  python -m venv msys_venv
  .\msys_venv\Scripts\pip install -r requirements.txt flask-login croniter
  ```

### 사례 2: Vendor JS 파일 손상
- **증상**: `SyntaxError: Invalid or unexpected token` (e.txt)
- **원인**: `.min.js` 파일에 주석 제거 정규식 적용
- **해결**: 원본 복구
  ```bash
  git checkout -- static/vendor/
  ```
